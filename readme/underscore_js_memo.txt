underscoreのメソッドについてのメモ

★_.extend(obj,source)
→オブジェクトを結合する
→これでいうとobjにsourceのオブジェクトを加える（コピーではない）
例）obj = {name:"hoge"}, source = {key:"foo}なら
obj = {name:"hoge", key:"foo"}となる

★_.template(templateString)
→htmlに用意したテンプレートを取得できる
例）
var compiled = _.template("hello: <%= name %>");
compiled({name: 'moe'});
=> "hello: moe"
$elでもできる
template: _.template($('#task-template').html())
→html上にclass=tsk-templateがあればそこを根として部分木を持ってくる

★_.debounce(function,milisecond)
→引数に指定したfunctionをmilisecondだけ遅らせて起動させるメソッド

★_.invoke(list,methodname)
→listのすべての値に対して第二引数で指定したメソッドを実行する
例）第二引数でdestroyとしたら、リスト内の値を全削除するみたいなことをするときに使用

★_.each(list or obj, function(val,key))
→listもしくはobjの入力値をfunctionに代入し、処理を実行する
listの場合は各列の値で行ってくれる
backbone.jsの中でlist or obj.each(function(val))で書くこともできる
