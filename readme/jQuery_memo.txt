jQueryの技術に関するメモ

★$(function(){});の意味
https://qiita.com/bakatono_super/items/fcbc828b21599568a597
→htmlは上から順番に実行されるため、途中でjQueryを呼んでしまうと中身の処理をする際に使用する、
変数や属性などの情報が何もないため、処理を実行できない。
そこで、htmlを全て読み込み終わった後に実行するために予約する意味を込めてこの書き方を行う。

★論理和の使い方
https://qiita.com/Imamotty/items/bc659569239379dded55
→var app = app || {}とあった時、appになにが入るか
appがfalseとみなせる値（0, -0, null, false, Nan, unidefined, "")以外ならappを、
そうでないなら{}を代入するという意味になる。

★last()メソッド
→指定したHTML要素の最後の要素を取得する
例えば、リストに対して実行すると最後のli要素が取得される。

★strictモード
→javascriptの冒頭部分で'use strict'と記述すると厳格モードが始まり、
より適格なエラーチェックが行われる。
そのため、コード内に存在する潜在的な問題を確認することもできる

★ENTER KEYの意味
→どのキーが押下されたかを判別することができる
例）window.event.keyCode
エンターキーは13となるため、13に等しければボタンクリックと同じ扱いができるため、
Todoリストのエンターキーでタスク追加などができる。
